*Data-driven product research with trend analysis, profit forecasting, and supply chain matching.*

An AI-powered product selection engine for e-commerce entrepreneurs and dropshippers. Combines market trend analysis, competition assessment, profit margin calculation, and supplier matching to help you find products with the highest chance of success.

## Features

| Feature | Description |
|---------|-------------|
| Trend Spotter | Identifies emerging product trends from social media, search data, and marketplace signals |
| Profit Calculator | Estimates profit margins including sourcing costs, shipping, fees, and advertising expenses |
| Competition Analyzer | Assesses market saturation, competitor pricing, and review landscapes |
| Supply Chain Matcher | Connects products to reliable suppliers with MOQ, lead time, and quality ratings |
| Niche Validator | Tests product-market fit with demand scoring, seasonality analysis, and risk assessment |
| Product Scorecard | Generates comprehensive product evaluation reports with go/no-go recommendations |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-ecommerce-selector-skill
```

## Usage

```
find products [category]   # Discover trending products in a category
evaluate [product]         # Full product viability analysis
calculate profit [product] # Detailed profit margin breakdown
find suppliers [product]   # Match with reliable suppliers
trending [platform]        # What's hot on Amazon, Shopify, TikTok Shop
```
